# Mundial 2026 — Grupos y Resultados

Página web sencilla para llevar los resultados del Mundial 2026 y ver cómo van quedando los 12 grupos.

- Escribe los marcadores de cada partido y la tabla de posiciones se ordena sola.
- Puntos, partidos jugados, goles a favor/en contra y diferencia de gol calculados al instante.
- Desempate: puntos → diferencia de gol → goles a favor.
- Resalta en verde el 1.º y 2.º (clasifican) y en ámbar el 3.º (opción de mejor tercero).
- Banderas de cada país al lado del nombre.
- Los nombres de equipos se pueden editar y todo se guarda en el navegador.

Hecho para usarlo fácil en celular o computador.
